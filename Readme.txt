Данная программа позволяет сравнить данные базы данных и с сайта https://www.list-org.com/ с помощью парсинга

Парсятся данные ИНН, названия, номера телефона, email.

Сравниваются данные по номеру телефона и в случае их несовпадения в определенную ячейку записываются данные о соответствии.

На выходе получаем клон базы данных, где и проверяется функционал.
